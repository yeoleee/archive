한국관광공사 내부 업무 시스템 메인 UX 구축
  1. 개요
  2. 한국관광공사 내부 업무 시스템 메인 UX 구축
  • Duration
    2024.07 - 2024.09
  • Type
    웹
  • Role
    메인 디자이너 (참여율 100%)
한국관광공사 그룹웨어 메인 대시보드의 UX 구조를 설계한 프로젝트다.

업무 흐름에 맞춰 핵심 기능과 정보를 재배치하고, 명확한 위계와 레이아웃 체계를 구축해 직관적인 메인 화면을 구현했다. 통일된 UI 기준을 적용하여 안정적이고 일관된 내부 업무 환경을 완성했다.
  1. 인사이트
  2. 기능을 더하기보다, 업무 효율을 높이는 명확한 구조로 다듬기
  • “너무 기능이 많고 정보가 분산되어 있어, 원하는 메뉴를 찾기가 어려워요."
  • “조금..세련되게 바꾸고 싶어요. 너무 오래됐잖아요?”
  • “제각각의 요소때문에 업무 집중도가 떨어지고 피로도가 누적되는 느낌이에요.”
따로 요구하는 추가 기능이 있는것도 아닌, 명확한 디자인 스타일이 있는 것도 아닌 상황이었다.

자주 사용하는 기능과 핵심 콘텐츠를 중심으로 메인 구조를 재설계하고, 고정 영역과 가변 영역을 명확히 구분해 시선 흐름을 정리했다.
한국관광공사 전용 컬러를 기능적 구분 요소로 활용하여 정보 위계를 강화하고, 버튼과 카드, 아이콘 스타일을 통일해 시스템 전반의 일관성을 확보했다.
