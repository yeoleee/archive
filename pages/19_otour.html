  2. 오투어 웹사이트 리뉴얼 참여 및 유지보수
  • Duration
    2023 - 2025
  • Type
    PC 웹, 모바일 웹
  • Role
    리뉴얼 참여 30%, 유지보수 참여 100%
오투어 웹사이트 리뉴얼 프로젝트에 참여하여 호텔 예약 메인 화면, 마이페이지, 모바일 메인 화면 및 임직원몰 UI 설계를 담당했다. 예약 흐름과 사용자 행동을 고려한 정보 구조 재정비를 통해 탐색과 결제 과정의 편의성을 강화했으며, 리뉴얼 이후에는 유지보수 전담으로 참여하여 기능 개선과 UI 안정화를 지속적으로 관리했다.
