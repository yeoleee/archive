  • Type

    메인 디자이너 (참여율 50%)
청소년과 보호자가 필요한 상담·복지 정보를 쉽고 빠르게 찾을 수 있도록 웹사이트를 구축했다. 사용자 니즈를 충족하는 직관적인 정보 구조를 설계하고, 콘텐츠 관리 효율성을 고려한 UI/UX 개선을 통해 센터의 아이덴티티를 유지하면서도 일관된 디지털 경험을 제공했다.

밝고 따뜻한 분위기의 비주얼과 안정감을 주는 컬러 체계를 적용하여, 청소년들이 편안함을 느낄 수 있는 온라인 공간을 구현했다.

  1. 인사이트
  2. 단순한 공공기관 홈페지지를 넘어 ‘신뢰의 공간’으로
청소년 상담 기관이기에 웹사이트는 단순한 정보 제공 채널이 아니라, 심리적으로 안전한 공간처럼 인식되면 좋겠다고 생각해서 딱딱한 행정 중심 구조가 아닌, 밝고 따뜻한 분위기와 친근한 비주얼을 통해 접근 장벽을 낮추는 것이 중요했다.

상담·지원 정보는 사용자가 빠르게 이해하고 접근 가능하게 정보 위계를 단순화하고, 직관적인 UI 구조를 설계해 사용자가 원하는 정보에 즉시 도달하도록 했다.
